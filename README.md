# Leax API

1. Clone project

2. ```yarn install```

3. Clone file ```.env.template``` and rename it to ```.env```

4. Set the enviroments variables 

5. Run BBDD
```docker-compose up -d```

6. Run the app in dev
```yarn start:dev```

7. Run Seed
```
http://localhost:3000/api/seed
```
